Detailed Description

RAII class that once acquired waits for the GPU to finish work up to this point, pauses the rendering, releases all internal resources from all graphics pipelines, then in destructor restores them.

Remarks
This can be useful when some render resource (like MSAA render target) has changed or about to be changed so that we can make sure all pipelines are refreshed to use the new/changed resource.
