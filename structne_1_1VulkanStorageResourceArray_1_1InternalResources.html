◆ activeSlots

std::unordered_set<VulkanStorageResourceArraySlot*> ne::VulkanStorageResourceArray::InternalResources::activeSlots

Set of slots that were inserted (equal to iSize).

Storing a raw pointer here because it's only used to update slot's index if the array was resized. Before the slot is destroyed it will be automatically removed from this set (see slot's destructor).

◆ iCapacity

size_t ne::VulkanStorageResourceArray::InternalResources::iCapacity = 0

The maximum number of elements that could be added to the array without expanding (recreating with a bigger size) the storage buffer.

◆ iNextFreeArrayIndex

size_t ne::VulkanStorageResourceArray::InternalResources::iNextFreeArrayIndex = 0

Index of the next free place in the array. Each new element inserted in the array will fetch this value (to be used) and increment it. Once this value is equal to iCapacity we will use noLongerUsedArrayIndices to see if any old indices are no longer being used.
